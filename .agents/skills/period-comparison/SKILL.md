# Period Comparison

## Purpose
Use this skill to compare finance metrics across two periods and explain the main drivers of change.

## When to use
Use this skill when the task involves:
- year-over-year comparison
- month-over-month comparison
- quarter-over-quarter comparison
- custom period comparisons

## Rules
- compare equivalent periods whenever possible
- report both absolute and percentage changes
- identify the categories driving the difference
- warn when prior-period coverage is incomplete
